Vagabulary - приложение, которое генерирует феминитивы.
Для создания алгоритма генерации использовались основные правила образования феменитивов, основанные на суффиксах мужских профессий.
На главной странице можно ввести свое слово в мужском роде. На его основе будут сгенерированы феминитивы. 
Если вас интересуют конкретные суффиксы у какого-то слова, можно воспользоваться фильтрами. Если выбранный суффикс не используется в 
образовании феминитива, результат об этом скажет. 
Также у приложения есть своебразный фидбэк. Если пользователь хочет предложить свой вариант феминитива, то он может отправить запрос с 
предложением на сервер.
В качестве дополнения на сайте есть игра, в которой можно проверить свое знание феминитивов. Она включает в себя 10 вопросов. Отвечая на
них пользователь зарабатывает баллы, которые суммируются. Результат можно узнать в конце теста.
